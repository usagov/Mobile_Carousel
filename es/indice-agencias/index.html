Un sitio web .gov pertenece a una organización oficial del Gobierno de Estados Unidos.

Los sitios web seguros .gov usan HTTPS
Un candado ( ) o https:// significa que usted se conectó de forma segura a un sitio web .gov. Comparta información sensible sólo en sitios web oficiales y seguros.

La Administración de Asuntos sobre la Vejez (AoA, sigla en inglés) es una agencia del Departamento de Salud y Servicios Sociales de EE. UU. cuya misión es desarrollar un sistema integral, coordinado y rentable de cuidado a largo plazo para promover un envejecimiento digno, activo e independiente. Para lograrlo, AoA dirige una de las redes nacionales más grandes de proveedores de cuidados en el hogar y las comunidades para las personas mayores y sus cuidadores.
